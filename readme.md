# ServerManage

A server manager that will be based on a html5 web server which connects to a nodeJS websocket on each server. ServerManage's feature list on first release will be Monitoring of System (CPU Usage, Disk Usage, Etc), Remote Terminal Access, Intergration with Dell's Drac, and much more.
